# ⏰ Alarm Clock using Python & Pygame

A simple yet functional command-line based Alarm Clock application built using Python and the Pygame library. This project allows users to set a custom countdown timer in minutes and seconds, after which an alarm sound is automatically played.

The project was developed to strengthen understanding of:
- Python fundamentals
- Functions and loops
- Time handling
- Real-time countdown logic
- Audio playback using Pygame
- File handling and project structure

This project demonstrates how core Python concepts can be combined to build a practical real-world application.

---

# 🚀 Features

- ⏳ Real-time countdown timer
- 🔊 Alarm sound playback using Pygame
- 🕒 Displays timer in `MM:SS` format
- 🔁 Live updating timer on the same terminal line
- ⚡ Lightweight and beginner-friendly project
- 🧠 Built using core Python concepts

---

# 🛠️ Technologies Used

- Python 3
- Pygame Library
- Time Module
- OS Module

---

# 📚 Concepts Used

This project demonstrates practical implementation of:

- Functions
- Loops (`for loop`)
- `range()` function
- `divmod()` for time conversion
- String formatting (`f-strings`)
- Escape characters (`\r`)
- Real-time execution using `time.sleep()`
- Audio playback using `pygame.mixer`
